# Phase Documentation Index

This directory is the scalable documentation structure for phase-by-phase project records.

## Purpose

`docs/phases/` is reserved for detailed session notes, phase reports, and evidence indexes organized by project phase.

This prevents current-state documents, archived documents, and per-phase working records from being mixed together.

## Structure

Each phase may contain:

- `session_notes/`: chronological notes from working sessions.
- `reports/`: formal phase or microphase reports.
- `evidence_index.md`: curated links to logs, experiments, or supporting artifacts.
